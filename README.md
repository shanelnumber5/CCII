GFC MSU Student Success Pathway Builder
README
Overview

The GFC MSU Student Success Pathway Builder is an interactive HTML prototype designed to support workshop discussions around student success, workforce readiness, belonging, and educational pathways.

The project was created as a presentation-ready deliverable for the SAVE Workshop and is intended to visually organize workshop findings, student barriers, institutional strengths, and strategic planning ideas into an engaging and accessible format.

Purpose

This prototype was designed to:

Present workshop findings in a visually engaging way
Move beyond static PDF documents and slide decks
Create a shared language around student success initiatives
Highlight barriers impacting student persistence and economic mobility
Connect coursework and student experiences to workforce competencies
Support future planning conversations around advising, belonging, and career readiness
