# GFC MSU Student Success Pathway Builder

Updated package with:
- Student Journey Framework
- Flip-card interactions
- Course-to-Resume Builder
- General Education categories and course translations
- Academic/workforce language toggle
- Reflection prompts
- Regular Mo logo used in visible page areas
- Superhero Mo reserved for the hidden easter egg
- Mo Tip callout using the regular mascot logo

## How to Use

Open `index.html` in a browser.

## Image Files Needed

Place these files in the `images` folder:

- `mo-logo.png`
- `mo-superhero.png`

## Easter Egg

Click the regular Mo logo in the header to open the hidden Captain Transferable Skills easter egg.
